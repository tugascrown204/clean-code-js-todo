# Clean Code JS Todo

## Overview
This is a simple to-do list application built using JavaScript, designed to showcase clean code principles. The purpose of this project is to provide a clear example of how to write maintainable and readable code while building an application from scratch.

## Features
- Add, edit, and delete tasks
- Mark tasks as complete
- Filter tasks by status (all, active, completed)
- Local storage for persistent data

## Clean Code Principles Demonstrated
- Meaningful naming conventions
- Single Responsibility Principle
- Proper use of comments and documentation
- Consistent indentation and spacing
- Avoidance of code duplication

## Getting Started
1. Clone the repository: `git clone https://github.com/yourusername/clean-code-js-todo`
2. Navigate into the directory: `cd clean-code-js-todo`
3. Open `index.html` in your browser.
